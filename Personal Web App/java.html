My Java App: The Rainbow Chooserizer

What does it do?

My app is an aid to the popular 5v5 shooter, Rainbow Six Siege. It solves the issue of being frustrated with having too many options with the game. Our chooserizer makes simple deicsions about strategies and operator picks to streamline the process. Above are sample pictures.

How to use the app

First, click the link below to access the repo. Second, clone the repo to your machine. Third, launch the Java project, and run the "Application.java". Four, Enjoy!
